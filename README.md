## How to Play

1. Run the program.
2. The game board will be displayed, with numbers 1-9 representing the available positions.
3. Players take turns entering a number to place their mark (X or O) in the corresponding position.
4. The game ends when a player wins by getting three of their marks in a row (horizontally, vertically, or diagonally), or when all positions are filled resulting in a draw.
5. To exit the game at any time, type 'e' and press Enter.

## Key Features

- Simple console-based user interface
- Two-player gameplay
- Win detection for rows, columns, and diagonals
- Option to exit the game at any time
